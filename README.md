# snake-game
It is a game in which user have to choose between snake,water,gun. Code will choose randomly. The total no. of attempt is 10. 
RULES:
YOU VS COMPUTER
GUN VS GUN = DRAW
GUN VS SNAKE = WIN
GUN VS WATER = LOSE
WATER VS WATER = DRAW
WATER VS GUN = WIN
WATER VS SNAKE = LOSE
SNAKE VS SNAKE = DRAW
SNAKE VS WATER = WIN
SNAKE VS GUN = LOS
